YGO 初動計算機

📄

PDF / XLSX をドラッグ＆ドロップで読み込み

またはクリックしてファイルを選択 / 下の表で手入力も可

先に「デッキ登録」タブでカードを登録してください
デッキと初動組み合わせを設定してから計算してください

全体の流れ

1
デッキを登録する

「デッキ」タブで、PDF / XLSX を読み込むか、手入力でカードを登録します。

PDFは 遊戯王ニューロン のマイデッキからダウンロードできます。

┌──────────────────────────┐ │ 📄 PDF/XLSX をドロップ │ │ または「+ カード追加」 │ │ │ │ 灰流うらら [−] 3 [+]│ │ 増殖するG [−] 1 [+]│ │ ... │ └──────────────────────────┘
2
初動の組み合わせを定義する

「組み合わせ」タブで、初動パターンを登録します。

各パターンは「この手札なら初動が成立する」という条件です。

群: 展開A ├─ カードX ×1以上 & カードY ×1以上 │ 群: 展開B ├─ カードZ ×1以上 │ 群: (未設定) ├─ カードW ×2以上

枚数の意味

  • 正の数 (1, 2, 3) → そのカードを N枚以上 引く必要がある
  • 0 → そのカードを 1枚も引かない ことが条件
  • 負の数 (-1, -2) → そのカードを |値+1|枚以下 しか引かないことが条件
    • -1 → 0枚以下（= 0と同じ）
    • -2 → 1枚以下

群について

群はパターンをグループ分けするためのラベルです。

  • 同じ群のパターンは OR で扱われます（いずれか1つ成立すればOK）
  • 群を設定しないパターンは自動的に「(未分類)」群にまとまります
3
初動確率を計算する

「初動計算」タブで手札枚数（先攻5枚 / 後攻6枚）を選んで「確率を計算」ボタンを押します。

計算結果の見方

┌─────────────────────────────┐ │ 89.23% │ ← 全パターン通しての確率 │ いずれかの初動を引く確率 │ └─────────────────────────────┘ ┌ 群ごとの確率 ───────────────┐ │ 展開A（5パターン） 72.15% │ ← 群内いずれか成立 │ 展開B（3パターン） 65.40% │ │ (未分類)（2パターン）38.20%│ └─────────────────────────────┘ ┌ 群の組み合わせ ─────────────┐ │ 展開A × 展開B AND 51.20% │ ← 両方同時に成立 │ 展開A × 展開B OR 86.35% │ ← どちらか成立 │ ... │ └─────────────────────────────┘
  • AND = 選んだ群の 全て が同時に成立する確率
  • OR = 選んだ群の いずれか が成立する確率

保存と読み込み

  • 💾 デッキを保存 → 名前を付けてサイドバーに保存（ブラウザに記憶されます）
  • ⬇ XLSX をダウンロード → Excel ファイルとして書き出し
  • XLSX 読み込み → このアプリで保存した XLSX をドロップすると復元できます
  • サイドバーの ✎ でデッキ名を変更、× で削除

キーボードショートカット

  • Ctrl + Z → 元に戻す（Mac: ⌘Z）
  • Ctrl + Y / Ctrl + Shift + Z → やり直す

対応ファイル

  • PDF → OCG公式デッキ登録シート。遊戯王ニューロンのマイデッキからPDFをダウンロードできます
  • XLSX → このアプリで出力したファイル（デッキ + 組み合わせを復元）

プライバシーについて

このアプリは完全にブラウザ内で動作します。入力したカード情報、アップロードしたPDF・XLSXファイルの内容は外部サーバーに一切送信されません。保存データはお使いのブラウザの localStorage にのみ保存され、第三者がアクセスすることはありません。
